# HavenRise (MVP)

Starter-Projekt: Anime-High-Fantasy City-Builder + Battles (Flutter)

## Steps to run locally (Android)
1. Install Flutter SDK on a PC/Mac.
2. `git clone <your-repo-url>`
3. `cd haven_rise`
4. `flutter pub get`
5. `flutter run` (with Android device/emulator)

## How to get on iPhone (TestFlight)
Follow the step-by-step instructions I gave earlier:
- GitHub -> push repo
- Codemagic -> connect repo -> configure App Store Connect API key -> start iOS build -> upload to TestFlight
